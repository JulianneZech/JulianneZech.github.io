Sample Class Projects


Artificial Intelligence

In this course, I used Java to implement five algorithms used in artificial intelligence applications: State Space Search, Bayesian Network, Theorem Prover, Incremental Reduced Error Pruning (IREP) Rule Learning, Q Learning.


Big Data Analytics

This class was a crash-course in using Apache Spark and Scala to process large amounts of data. I implemented a version of Google's PageRank algorithm on the entire ~70 GB English Wikipedia corpus. I also wrote code to classify articles as sports or non-sports related and computed article word counts with a focus on using efficient operations.


Data Mining

In this course, I implemented Naive Bayes, Neural Network, and Association Rule Mining in Python from scratch, i.e. without using any machine learning packages or libraries. I tested the first two classifiers on Census data and the third on customer purchase data and was graded on both accuracy and efficiency.


Network Security

The main project in this class was an Secure Instant Messenger Application that uses threading to send encrypted and authenticated messages to an unlimited number of other clients. I used Google Cloud protocol buffers in my design and tested the application on Google Cloud virtual machines.
